# Quran KRD

A React-based web application designed to provide the Quran with Kurdish localization. This project features a custom synchronization system that precisely aligns localized Kurdish text with specific audio segments for a seamless reading and listening experience.

## ✨ Features

* **Audio-Text Synchronization:** Real-time alignment of Quranic recitation audio with the corresponding translated text.
* **Kurdish Localization:** Accessible interface and translations tailored for Kurdish readers. 
* **Responsive Design:** A clean, user-friendly interface built to work seamlessly across desktop and mobile devices.
* **Efficient Data Handling:** Fast rendering of chapters and verses using optimized JSON data structures.

## 🛠️ Built With

* **Frontend:** React, JSX, HTML, CSS
* **Data & API:** JSON, REST API concepts
* **Version Control:** Git, GitHub
